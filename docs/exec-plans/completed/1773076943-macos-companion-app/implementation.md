## Summary

Full implementation of native macOS companion app with feature parity to `companion/pixel_agents_bridge.py`:

- **Model**: Direct port of Python business logic (ProtocolBuilder, StateDeriver, AgentTracker, TranscriptWatcher, UsageStats)
- **Transport**: Serial via POSIX fd with IOKit USB detection; BLE via CoreBluetooth NUS client with PIN-based device selection
- **Services**: BridgeService orchestrator with 4 timers (poll 250ms, heartbeat 2s, usage 10s, reconnect 2s); ScreenshotService for serial-only screenshot capture
- **Views**: SwiftUI MenuBarExtra popover with connection status, transport picker, agent list, usage bars, screenshot button, launch at login, quit
- **App lifecycle**: LSUIElement (no Dock), sleep/wake handlers, SMAppService for login item

No deviations from plan. All planned files created and functional.

## Audit Fixes

Fixes applied from the 7-subagent post-implementation audit:

1. **[FIXED] QA/Interface/State #1: Agent state/toolName not written back to tracker** — Added `a.state = state` and `a.toolName = tool` to the `tracker.update()` call in `processTranscripts()`. Also write back `hadToolInTurn`/`activeTools` on the else branch when derive returns nil.

2. **[FIXED] State #2: setTransport() guard bug — transport never switches** — Replaced `$bridge.transportMode` Picker binding + `.onChange` with a custom `Binding(get:set:)` that calls `setTransport()` directly. Removed redundant `.onChange`.

3. **[FIXED] Resource/Security #3: SerialTransport fd data race** — Added `fdLock: NSLock` to protect all `fd` reads/writes. `send()`, `handleRead()`, `disconnect()`, and `isConnected` all acquire the lock before accessing `fd`.

4. **[FIXED] Security #4: Partial write not handled** — Changed `send()` to loop until all bytes are written or an error occurs.

5. **[FIXED] QA/Security/Resource #5: TranscriptWatcher missing deinit** — Added `deinit { stopMonitoring() }` to prevent dangling FSEvents pointer.

6. **[FIXED] QA/Security #6: Unbounded RLE pixel growth** — Clamped run count: `min(runCount, remaining)` where `remaining = totalPixels - pixels.count`.

7. **[FIXED] DX #7: Dead code** — Removed unused `import Combine`, `lastConnectedDeviceID`, `onDataReceived`, `msgStatusText`.

8. **[FIXED] Security #8: Int(minutes) potential trap** — Changed to `Double` clamping: `max(0.0, min(minutes, Double(UInt16.max)))` then `UInt16(clamped)`.

9. **[FIXED] Security #9: fileOffsets grows unboundedly** — Added pruning in `findActiveTranscripts()`: `fileOffsets = fileOffsets.filter { activePaths.contains($0.key) }`.

10. **[FIXED] QA/DX #10: ISO8601DateFormatter recreated each call** — Cached as `static let` properties.
