## Decision Records

When making significant architectural decisions, create a decision record:

**File naming:** `docs/design-docs/YYYY-MM-DD-<slug>.md`

**Template:**

```markdown
# Decision: {Title}

**Status:** Draft | Active | Superseded by {link}
**Date:** YYYY-MM-DD

## Problem
What problem does this decision address?

## Decision
What was decided and why?

## Alternatives Considered
What other options were evaluated?

## Consequences
What are the trade-offs and implications?
```
