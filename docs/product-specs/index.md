# Product Specifications

Catalog of user-facing behavior specifications.

## Core User Flow

1. **Flash firmware** to ESP32 via USB (PlatformIO or browser flasher)
2. **Connect companion** — run Python bridge (`python3 run_companion.py`) or macOS app
3. **Use CLI normally** — Claude Code, Codex CLI, Gemini CLI, or Cursor
4. **Watch agents animate** — characters walk to desks, type, read, idle in the office scene
